# Wallpaper
Windows下的动态壁纸软件

## 开源协议
本代码遵守GPLv3.0开源

## 使用
安装pywin32模块
```
pip install pywin32
```
在`config.json`中的`video`键的值输入视频路径(需要2个反斜杠)。

## 注意
程序运作是可能会改变缩放率，忽略即可，程序运行要十秒左右的时间，最后啥都不干
